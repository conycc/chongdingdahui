# chongdingdahui
冲顶大会答题帮助脚本

> 这只是个帮助脚本，作为参考，没有准确答案。**多读书才是王道！！！**

### 原理

1. 获取题目及选项数据
2. 拼接搜索关键词，自动打开浏览器搜索
3. 打印各个选项并返回(问题+选项)在微软必应的搜索返回结果数作为参考，非正确答案
4. 得到结果，按任意键继续

### 使用帮助

脚本运行环境：
```
1. mac系统
2. Python3.6.3
```

使用项目：
```
pip install -r requirements.txt
python work.py
```
